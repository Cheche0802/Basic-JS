'use strict'
Obliga a declarar variables antes de usarlas.
No permite la asignación a variables no declaradas.
Prohíbe el uso de variables llamadas "eval" y "arguments" como nombres de parámetros o variables locales.
Hace que "this" sea undefined en funciones que no son métodos de un objeto.
No permite la duplicación de nombres de parámetros en funciones.
Prohíbe la eliminación de variables, funciones o argumentos de funciones.
No permite usar palabras reservadas como nombres de variables (por ejemplo, "let", "class", "enum").
Hace que el uso de "with" genere un error.
Prohíbe la asignación a solo lectura (por ejemplo, a propiedades de solo lectura de objetos).
Hace que las asignaciones a propiedades y objetos congelados generen un error.
